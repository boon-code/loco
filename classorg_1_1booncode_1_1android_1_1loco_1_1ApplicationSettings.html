Detailed Description

This class manages all (persistent) settings from this application.

To create this object, a context of this application is necessary to retrieve the application context, that is used to get an isntance of the default SharedPreferences. This instance is then used to save and load simple data elements as strings and integers...

Note:
Note that some keys of this class (KEY_SIM_PROTECT, KEY_THEFT_MODE) shouldn't be changed because they are related to res/xml/preferences.xml used by SettingsActivity.
