Detailed Description

Settings Activity, this activity automatically saves changes to the default SharedPreference.

Currently only 2 Options are included:

  • Theft-Mode: Currently just a flag that will be set to true if SimCheckingService detects a different sim card.
  • Sim-Protection: If this flag is set to true, sim-protection is enabled every time the device is booted.
See also:
res/xml/preferences.xml for the preference options.
